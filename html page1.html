Introduction to HTML

HTML (Hypertext Markup Language) is the standard language used to create webpages. It defines the structure of a webpage using markup tags.

HTML is the building block for web development. Every webpage you see on the internet is created using HTML.

HTML Structure

A basic HTML document has the following structure:

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

Explanation:

  • <!DOCTYPE html>: Defines the document type and version of HTML.
  • <html>: Root element that wraps the entire HTML document.
  • <head>: Contains metadata like title, links to stylesheets, etc.
  • <body>: Contains the visible content of the webpage.

Common HTML Tags

Here are some common HTML tags:

  • <h1> to <h6>: Headings, used for titles and subtitles.
  • <p>: Paragraph tag, used to create text blocks.
  • <a>: Anchor tag, used for links. Example: <a href="https://www.example.com">Click me</a>
  • <img>: Image tag, used to display images. Example: <img src="image.jpg" alt="Description">
  • <ul>: Unordered list, used for bulleted lists. Example: <ul><li>Item 1</li></ul>

HTML Forms

Forms are used to collect user input. Here is an example:

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

Embedding Multimedia

HTML allows you to embed images, audio, and video. Here are examples:

  • Image: <img src="image.jpg" alt="Description">
  • Audio: <audio controls><source src="audio.mp3" type="audio/mp3"></audio>
  • Video: <video controls><source src="video.mp4" type="video/mp4"></video>

Advanced HTML Topics

As you progress, you can learn more advanced HTML topics such as:

  • Forms and Input Types: Creating complex forms with different input fields.
  • SEO Optimization: Using meta tags, header tags, and sitemaps for better search engine visibility.
  • HTML5 Semantic Tags: New tags like <article>, <section>, <nav>, <header>, and <footer>.
  • HTML APIs: Using APIs like Geolocation, Canvas, Web Storage, etc.
