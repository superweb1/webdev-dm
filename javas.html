Lessons

Lesson 1: Variables and Data Types

In JavaScript, you can store values using variables. Variables can hold different types of data:

  • String: Text (e.g., "Hello, World!")
  • Number: Numeric values (e.g., 42, 3.14)
  • Boolean: True or false (e.g., true, false)
  • Object: A collection of key-value pairs (e.g., {name: "John", age: 30})
  • Array: An ordered list of values (e.g., [1, 2, 3])
  • Null: Represents an intentional absence of a value
  • Undefined: A variable that has been declared but not yet assigned a value

Example of declaring a variable:

let name = "John";

Lesson 2: Functions

Functions in JavaScript are blocks of code that are designed to perform a particular task. They can be invoked (called) by their name.

  • Function Declaration: function myFunction() { }
  • Function Expression: const myFunction = function() { }
  • Arrow Function: const myFunction = () => { }

Example:

function greet(name) {
    return "Hello, " + name + "!";
}

This function takes one parameter, name, and returns a greeting.

Lesson 3: Loops

Loops allow you to execute a block of code multiple times. There are different types of loops:

  • for loop: Used for running a block of code a specific number of times.
  • while loop: Runs as long as a specified condition is true.
  • do...while loop: Executes code at least once and then continues based on a condition.

Example of a for loop:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

This will log numbers from 0 to 4 in the console.
